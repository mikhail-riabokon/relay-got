# Relay + GraphQL example

This is an [example](http://relay-got.herokuapp.com/) of react-relay + graphql usage. Data was taken from Game of Thrones world.

[Presentation (Ru) ](https://www.youtube.com/watch?v=CePdnxcnnCA)

## Development (MacOSX)
* ```npm install```
* ```npm run dev```
